## ✨ Features

- 🧾 **OPay-style receipt layout** — familiar, professional transaction receipt design
- ✏️ **Fully editable fields** — name, amount, reference, date, status — all in-browser
- 📄 **Instant PDF download** — one click, no server, no email
- 🔒 **Privacy-first** — all data stays in your browser, nothing is sent anywhere
- 📱 **Mobile-ready** — works on phone just as well as desktop
- ⚡ **Zero dependencies** — pure HTML/CSS/JS, loads instantly

---

## 🚀 Quick Start

```bash
git clone https://github.com/jayblast-spec/receipt-generator.git
cd receipt-generator
# Open index.html directly in your browser — no build step needed
open index.html
```

Or visit the [live demo](https://receipt-generator-eosin.vercel.app) instantly.
